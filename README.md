# Smart Loan Eligibility and Risk Assessment

A full-stack AI-powered Loan Management System that evaluates customer loan eligibility using Machine Learning.
Built with Next.js (Frontend) and Django + Django REST Framework (Backend) using PostgreSQL as database.


## Project Overview

##### The Smart Loan Eligibility System helps:

1. Customers apply for loans online

2. Officers review and approve/reject loans

3. System automatically calculate risk score using ML model

4. Manage loan applications efficiently

##### The system predicts loan approval probability based on:

1. Credit score

2. Income

3. Employment status

4. Previous loan history

5. Financial patterns
